Perguntas Frequentes

1. Oque é HTML?

HTML é alinguagem padrão para criação de páginas web.Ele define a estrutura eo conteúdo de uma página.

2. O que são elementos HTML?

Elementos HTML são as partes que compõem uma página web,como títulos,parágrafos,imagens,links,entre outros.

3. Como criar uma lista não ordenada em HTML?

Para criar uma lista não ordenada em HTML,você pode usar o elemento <ul> (unorderedlist) para criar a lista e <li> (list item) para criar cada item da lista.
